# Angular seed project

`Angular 1.x` `webpack` `es6` 

A complete, yet simple, starter for Angular 1.x using Webpack.


### Quick start

```
# install the dependencies with npm
$ npm install

# start the server
$ npm start

```

go to [http://localhost:8080](http://localhost:8080) in your browser.
